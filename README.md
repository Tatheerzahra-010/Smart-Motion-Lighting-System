# 💡 Smart Motion-Based Lighting System with Automatic Night Door

## 📖 Description
This project is based on an automated system that controls lighting and door operation using motion detection, especially during nighttime.

## 🎯 Objectives
- To save energy using motion sensors  
- To automate lighting system  
- To improve security
